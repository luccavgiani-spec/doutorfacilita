# Plantão Digital

Plataforma de telemedicina pronto-atendimento. Paciente paga, entra na fila em tempo real, é atendido por médico via vídeo em até 10 minutos.

## Infraestrutura

| Camada | Provedor |
|---|---|
| Repositório | GitHub (privado) |
| Deploy | Vercel |
| Backend / Banco / Auth / Realtime | Supabase (sa-east-1) |
| Desenvolvimento assistido | Claude + Claude Code |

## Stack de aplicação

Ainda não decidida. Frontend, framework, integrações de vídeo, pagamento e demais escolhas serão definidas em sessões futuras.

## Compliance

- CFM Resolução 2.314/2022
- NGS2 / SBIS (atendimento técnico)
- LGPD
- Retenção de prontuários: 20 anos

## Time

Desenvolvido por Nó Tech Stack para os 3 médicos sócios.
